# Telegram Unblock Monitor

Мониторинг разблокировки пользователей в Telegram. Отправляет уведомление в Избранное при разблокировке.

## Быстрый старт

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите: `python main.py`
4. При первом запуске скрипт попросит:
   - API ID и API Hash (получить на https://my.telegram.org/apps)
   - Username'ы пользователей для мониторинга
   - Номер телефона и код из Telegram для авторизации

## Получение API ключей

1. Перейдите на https://my.telegram.org/apps
2. Войдите с номером телефона
3. Создайте приложение
4. Скопируйте API ID и API Hash

## Как работает

- Проверяет доступность профиля каждые 3 секунды
- Когда пользователь достаёт вас из ЧС - профиль становится доступным
- Отправляет уведомление в Избранное в Telegram
- Работает 24/7 в фоне

## Настройки

После первого запуска создаётся файл `config.json` с настройками:
- `api_id` - ваш API ID
- `api_hash` - ваш API Hash  
- `target_users` - список пользователей для мониторинга

Чтобы изменить настройки - отредактируйте `config.json` или удалите его и запустите скрипт заново.

## Деплой на Render.com (бесплатный хостинг 24/7)

1. Форкните этот репозиторий
2. Зайдите на https://render.com
3. Создайте новый **Background Worker**
4. Подключите ваш форк репозитория
5. Deploy

При первом запуске на Render нужно будет зайти в логи и ввести данные для настройки.

## Файлы

- `main.py` - основной скрипт
- `requirements.txt` - зависимости Python
- `config.json` - конфигурация (создаётся автоматически, не коммитится)
- `monitor_session.session` - сессия Telegram (создаётся автоматически, не коммитится)
